	突然有一天脑中闪过一个念头，就把它做出来了。当然并没有想过去入侵别人的电脑或者从中获取什么利益，只是当作恶搞朋友的一个小伎俩罢了。请误将此作品用于商业或者其余任何有害于他人利益的地方（当然，恶搞朋友除外，这也是他诞生的目的，哈哈）。下面介绍一下他其中用到的知识：
	主要思想就是就是利用两个部分相互关联，当你试图去关闭一个界面的时候，另一个界面就会启动，并且覆盖你的屏幕。当然这对于一些杀毒软件简直在简单不过了，分分钟将软件扼杀在摇篮里，所以只用于朋友。
	要想要关闭这个，有两种方法。第一，你的手速足够快，可以使用Alt+F4关闭，按键速度要比软件之间的相隔时间快就好了。第二，直接使用Ctrl+Shift+Alt+Z,同时按下，就可以关闭了，这相当于密钥吧。
